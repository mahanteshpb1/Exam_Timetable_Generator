How to Use

This section will explain how to use the Automated Time-Table Scheduler.

Currently, the application supports generating timetables using a pre-defined input.txt file.

Steps:

  1. Ensure the input.txt file is located in the root directory of the application with your desired student group and teacher data.
  2. Navigate to the timetable generation page.
  3. Click the "Generate Timetable" button to process the input and display the generated timetable.
